🚀 Overview

Data Mining คือกระบวนการค้นหา รูปแบบ (patterns), ความสัมพันธ์, หรือ ความรู้ที่มีประโยชน์ จากข้อมูลจำนวนมาก โดยใช้เทคนิคจากหลายสาขา เช่น

Machine Learning , สถิติ (Statistics) , ฐานข้อมูล (Database Systems) ,การรู้จำรูปแบบ (Pattern Recognition)


เป้าหมายคือ เปลี่ยนข้อมูลดิบจำนวนมหาศาลให้กลายเป็นข้อมูลเชิงลึกที่ใช้งานได้ (actionable knowledge)
เช่น การทำนายพฤติกรรมลูกค้า, ตรวจจับการทุจริต, แนะนำสินค้า หรือค้นหาความผิดปกติของข้อมูล
